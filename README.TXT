INANIS
------
Word processor for vapor people.
Intended for stream of consciousness sessions and dynamic poetry.	
It's main features revolve around net arty visuals and relaxing sounds that evolve from your typing 
style and words and the possibility of record your sessions or phrases and play them back live all at once. 
Online multiwriter has been trunkated for now.

INSTALLATION
------------
1 - Download the correct version for your OS from https://bitbucket.org/jerebab/inanis/downloads or http://jeremiasbabini.com/inanis.php
2 - Extraxt zipzile
3 - Run Inanis.exe

MISCELLANEOUS KEYBOARD SHORTCUTS
--------------------------------
[↑] Move up the sea
[↓] Move down the sea
[-] Decrese font size
[+] Increase font size
[Ctrl]+[S] Save all writings to a .txt file 
[Ctrl]+[E] Export an image file of the screen
[Esc] Close aplication
